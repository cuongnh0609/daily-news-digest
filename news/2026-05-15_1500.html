B1AWS Bedrock Agent Registryで MCP/Skill/A2Aエージェントを動的ロード——コードデプロイ不要で新ツールを即時反映

📰 本文 + 和越対訳

Amazon Bedrockに組み込まれた「Agent Registry」を活用し、MCPサーバー・Agent Skills・A2Aエージェントを動的に検索・注入するアーキテクチャが注目を集めている。このシステムは、エージェント起動時にたった1つのツール（Registryの意味検索）だけを保持し、ユーザーのリクエストに応じて関連リソースを動的に発見してエージェントへ追加するという設計だ。カスタムフックの「RegistryLazyLoader」が検索結果を受け取り、リソース種別に応じてMCPツール・スキルプロンプト・A2Aエージェントラッパーをリアルタイムで注入する。さらに詳細なJSON応答をコンパクトな要約に書き換えることで、トークン効率を維持しながら多彩な能力を持つエージェントを実現している。この「必要なときだけロードする」原則はAnthropicが推奨するプログレッシブディスクロージャーの思想と一致しており、組織内で新しいリソースをRegistryに登録するだけで、コードの再デプロイなしに全エージェントがその機能を即時利用できる。

Kiến trúc sử dụng AWS Bedrock Agent Registry để tìm kiếm và nạp động các MCP server, Agent Skills, và A2A agent đang thu hút sự chú ý. Hệ thống khởi chạy chỉ với một công cụ duy nhất là tìm kiếm semantic trong Registry, rồi khám phá và thêm các tài nguyên phù hợp theo yêu cầu của người dùng. Hook tùy chỉnh "RegistryLazyLoader" nhận kết quả tìm kiếm, tự động inject MCP tool, Skills prompt hoặc A2A agent wrapper theo loại tài nguyên. Phản hồi JSON dài được viết lại thành tóm tắt ngắn gọn để tối ưu token. Nguyên tắc "chỉ load khi cần" này phù hợp với triết lý progressive disclosure của Anthropic — và khi tổ chức đăng ký tài nguyên mới vào Registry, tất cả agent có thể sử dụng ngay mà không cần redeploy code.
