B1AWS、インストール不要のブラウザ版AIコーディングエージェント「Kiro Web」をプレビュー公開――仕様駆動で自律pull requestも

📰 本文 + 和越対訳

Amazon Web Servicesは、Webブラウザだけで利用できるAIコーディングエージェント「Kiro Web」をプレビュー公開した。ローカルへのインストールが不要で、GitHubリポジトリを選択するだけで即座に利用を開始できる点が特徴だ。動作モードは「インタラクティブモード」と「自律モード」の2種類が用意されており、後者ではエージェントが要件の確認から実装計画の策定、コーディング、プルリクエストの生成まで一連の作業を自動で完結させる。Kiro Webは仕様書を先に定義してから実装を進める「Spec-Driven Development」のアプローチを採用しており、テスト設計も仕様に基づいて行われる。Kiro IDEやKiro CLIと組み合わせることで、複数の開発環境を横断しながら一貫したエージェント体験が得られる設計になっている。AWSはこのエコシステム全体への自律型ワークフローの展開を順次進める方針だ。

Amazon Web Services đã public preview Kiro Web — một AI coding agent chạy hoàn toàn trên trình duyệt mà không cần cài đặt local. Người dùng chỉ cần chọn GitHub repository là có thể bắt đầu ngay. Công cụ này hỗ trợ hai chế độ: "Interactive Mode" (cộng tác với AI từng bước) và "Autonomous Mode" (AI tự động hoàn thiện từ xác nhận yêu cầu, lập kế hoạch, code đến tạo pull request). Kiro Web áp dụng phương pháp "Spec-Driven Development" — định nghĩa specification trước rồi mới implement, với test design cũng dựa trên spec. Kết hợp với Kiro IDE và Kiro CLI, hệ sinh thái cho phép trải nghiệm agent nhất quán trên nhiều môi trường phát triển khác nhau.
