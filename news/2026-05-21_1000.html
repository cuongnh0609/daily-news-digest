B-01Claude Codeに永続記憶を持たせるライブラリを自作——FTS5+ベクトル検索で1ヶ月1239件の記憶を管理

📰 本文 + 和越対訳

SonicGardenのエンジニアが、Claude Codeにセッションをまたいだ永続記憶機能を持たせるPythonライブラリ「claude-memory」を自作し、その詳細をZennで公開した。Claude Codeはデフォルトでセッションが終わると文脈が失われるが、このライブラリはClaude Code Hooksを活用してプロジェクト固有の情報（命名規則、設計方針など）を保存・検索できるようにする。検索アーキテクチャはSQLiteのFTS5（トライグラム全文検索）とsqlite-vec（ベクトル類似検索）を組み合わせ、Reciprocal Rank Fusion（RRF）で結果を統合する。埋め込みモデルには日本語特化の「Ruri v3-310m」（310Mパラメータ、768次元）を採用し、HF_HUB_OFFLINE=1でプライバシーを確保しながら完全ローカルで動作する。記憶データはJSONLをソースオブトゥルースとしGitで複数マシン間の同期も可能だ。1ヶ月間の運用実績では1239件の記憶、23プロジェクト、DBサイズ26MBを達成しており、Claude Code組み込みの自動メモリとも共存できる。

Một kỹ sư tại SonicGarden đã tự xây dựng thư viện Python có tên claude-memory để trao cho Claude Code khả năng lưu trữ bộ nhớ liên tục xuyên suốt các phiên làm việc, và chia sẻ chi tiết lên Zenn. Claude Code theo mặc định sẽ mất ngữ cảnh khi kết thúc phiên, nhưng thư viện này tận dụng Claude Code Hooks để lưu và tìm kiếm thông tin đặc thù của từng project (quy ước đặt tên, chiến lược thiết kế...). Kiến trúc tìm kiếm kết hợp FTS5 của SQLite (tìm kiếm toàn văn trigram) với sqlite-vec (tìm kiếm tương đồng vector), tổng hợp kết quả bằng Reciprocal Rank Fusion (RRF). Mô hình embedding sử dụng "Ruri v3-310m" chuyên tiếng Nhật, chạy hoàn toàn offline để bảo mật. Sau 1 tháng vận hành, hệ thống đã tích lũy 1.239 ký ức, 23 project, với DB nặng 26MB và có thể đồng bộ qua Git giữa nhiều máy.
