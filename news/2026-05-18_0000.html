B-01OpenAI vs Anthropic——AI競争の主戦場はモデル性能から「ハーネス」へ移行した

📰 本文 + 和越対訳

AIコーディングエージェントの競争軸がベンチマーク性能から「ハーネス」へと移り変わったという分析が注目を集めている。ここでいう「ハーネス」とは、AIモデルを包む実行環境全体のことで、CLI/IDE統合・ファイル編集・コマンド実行・Git/GitHub連携・MCPサーバー接続・パーミッション管理・CLAUDE.mdなどの設定ファイルが含まれる。AnthropicはClaude CodeとMCPでこのハーネス戦略を先行し、MCPはLinux Foundationに寄贈された後、ChatGPT・Cursor・Gemini・VS Codeなどが採用、公開MCPサーバー数は1万を超えた。OpenAIは対抗策として、Codex（クラウドエージェント）・AGENTS.md・Responses APIへのMCP対応などを展開。Anthropicの年換算収益は2025年末の約90億ドルから2026年4月には300億ドル超に急増しており、ハーネスの優位性が実際のビジネス成果に直結していることが示されている。

Một phân tích đáng chú ý cho rằng cuộc cạnh tranh giữa các AI coding agent đang chuyển trọng tâm từ điểm benchmark sang "harness" — toàn bộ môi trường thực thi bọc quanh mô hình AI, bao gồm tích hợp CLI/IDE, chỉnh sửa file, thực thi lệnh, kết nối Git/GitHub, MCP server, quản lý quyền, và các file cấu hình như CLAUDE.md. Anthropic đi tiên phong chiến lược harness với Claude Code và MCP; sau khi MCP được hiến tặng cho Linux Foundation, ChatGPT, Cursor, Gemini và VS Code đều đã áp dụng, với hơn 10.000 MCP server công khai. OpenAI phản công bằng Codex (cloud agent), AGENTS.md và hỗ trợ MCP trong Responses API. Doanh thu quy năm của Anthropic tăng vọt từ ~9 tỷ USD cuối 2025 lên hơn 30 tỷ USD vào tháng 4/2026, cho thấy lợi thế harness đang trực tiếp chuyển hóa thành kết quả kinh doanh.
