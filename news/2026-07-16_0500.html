B2Google Cloud、生成AIコードを瞬時に隔離実行「Cloud Runサンドボックス」をパブリックプレビュー公開

📰 本文 + 和越対訳

Google Cloudは、生成AIが生成した信頼できないコードを安全に隔離して実行できる「Cloud Run Sandboxes」（Cloud Runサンドボックス）をパブリックプレビューとして公開した。生成AIがタスクをこなす際にコードを生成・実行したり、ヘッドレスブラウザ経由でWebにアクセスしたりする場面が増えるにつれ、そのコードを既存のインフラから分離して実行する需要が高まっている。Cloud Runサンドボックスは、そうした信頼できないコードのための隔離された実行環境を数秒以内に起動できる点が特徴だ。既存のCloud Runと同じAPIを使えるため移行コストが低く、セキュリティと開発者体験の両立を狙った設計となっている。エージェントインフラの観点から、コード実行サンドボックスはAIエージェントの安全なツール使用において極めて重要な役割を担うとされている。

Google Cloud đã công bố bản preview công khai của "Cloud Run Sandboxes" — môi trường thực thi cô lập cho phép chạy an toàn code được tạo bởi AI sinh tạo. Khi các AI agent ngày càng tạo ra và chạy code, truy cập web qua headless browser, nhu cầu cô lập code không đáng tin cậy khỏi hạ tầng hiện có ngày càng tăng. Cloud Run Sandboxes có thể khởi động môi trường cô lập trong vài giây và sử dụng API tương tự Cloud Run thông thường, giúp giảm chi phí di chuyển. Từ góc độ hạ tầng agent, sandbox thực thi code đóng vai trò cực kỳ quan trọng trong việc đảm bảo an toàn khi agent sử dụng công cụ.
