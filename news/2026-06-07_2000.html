B-01AI駆動開発はここまで来た——Claude × Figma MCP で1日アプリ制作、デザインと実装を同時進行

📰 本文 + 和越対訳

Zenn に投稿されたエンジニアの記事によると、Claude と Figma の Model Context Protocol（MCP）を組み合わせることで、デザインから実装までを1日で完成させる開発ワークフローが実現できるという。著者は「PetSpot」というペット向けスポット検索アプリを実際に1日で構築した経験をもとに、具体的なセットアップ手順を公開した。Figma MCP を導入すると Claude がデザインの構造（余白・コンポーネント・レイアウト）を直接読み取れるため、Figma の開発モードで「Copy example prompt」を実行するだけで、そのままコード実装に移行できる。従来の「デザイン→仕様書→実装→差分修正」というサイクルが大幅に短縮され、デザイン変更への追従コストも激減した。ただし必要要件として、macOS 環境・Claude CLI/Code 導入済み・Figma Pro 以上のプラン・Dev Mode の有効化が必要とされる。

Theo bài viết trên Zenn, việc kết hợp Claude với Figma qua Model Context Protocol (MCP) cho phép hoàn thành toàn bộ quy trình từ thiết kế đến lập trình chỉ trong một ngày. Tác giả đã thực tế xây dựng ứng dụng tìm kiếm địa điểm cho thú cưng "PetSpot" và chia sẻ chi tiết quy trình. Khi tích hợp Figma MCP, Claude có thể đọc trực tiếp cấu trúc design (margin, component, layout), cho phép chuyển thẳng sang implementation chỉ bằng cách sao chép prompt từ chế độ Development của Figma. Chu trình truyền thống "design → spec → implementation → fix diff" được rút ngắn đáng kể, chi phí đồng bộ hóa khi design thay đổi cũng giảm mạnh. Yêu cầu: macOS, Claude CLI/Code đã cài, Figma Pro trở lên, và Dev Mode được kích hoạt.
