OpenRouter has released Fusion, a multi-model deliberation system designed to improve accuracy on research and complex reasoning tasks. Rather than routing a prompt to a single model, Fusion sends the request to a panel of expert models in parallel—each with web search and web fetch enabled—then passes all results to a judge model that synthesizes a final response. The synthesis explicitly addresses consensus points, contradictions, partial coverage, unique insights, and blind spots across the panel's outputs. Fusion ships with two presets: Quality (default, higher accuracy) and Budget (lower cost). Advanced users can customize which analysis models form the panel and which model serves as the judge. The API uses the same interface as standard OpenRouter completions, making adoption straightforward for existing integrations. Fusion is positioned as an alternative to repeated single-model queries for tasks where accuracy matters more than latency or cost.
