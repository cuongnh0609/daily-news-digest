B-1「うちのAIは喋るぞ」——Claude CodeとCodexに音声通知を与えるMCPサーバー「speak-mcp」を作って公開

📰 本文 + 和越対訳

複数のAIエージェントを並列で動かしていると、各ターミナルを常に監視し続けなければならないという課題がある。この問題を解決するために開発されたのが、MCP（Model Context Protocol）サーバー「speak-mcp」だ。Claude Code、Codex CLI、ChatGPTなどのAIツールがTTSエンジンを経由して音声でアウトプットを読み上げることができるようになる。設定はCLAUDE.mdで行い、「どのタイミングで喋るか」「どのスタイルで話すか」「どのTTSバックエンドを使うか」を柔軟に制御できる。想定される主なユースケースは3つ——タスク完了時の告知、エージェントが詰まった際の質問呼びかけ、そして複数エージェントへの一斉ステータス通知だ。TTSバックエンドとして、Gemini TTS（自然さ最高、StylePrompt対応）、irodori-TTS（音声クローニング）、ElevenLabs、GPT-4o mini TTS、Grok TTSが比較検証されており、用途に応じて選択できる。テキストログと音声出力の二重保持により、出力の検索性も維持されている。開発者によれば、音声でやり取りすることで「AIパートナーとの協働感」が高まり、長時間のコーディングセッションへの没入度が上がったという。リポジトリはgithub.com/mimo-3/speak-mcpで公開中だ。

Khi chạy nhiều AI agent song song, developer phải liên tục theo dõi nhiều terminal cùng lúc — đây là vấn đề "speak-mcp" ra đời để giải quyết. C-1The Agent Development Lifecycle: Harrison Chase's Four-Phase Framework for Production AI Agents

📰 Article (English)

LangChain CEO Harrison Chase published a comprehensive framework for building and operating AI agents in production environments — the Agent Development Lifecycle. The framework comprises four phases with Governance as a cross-cutting concern throughout. In the Build phase, practitioners choose between code-first frameworks (LangChain, CrewAI, Claude Agents SDK), stateful runtimes (LangGraph), or low-code platforms (n8n). The Test phase requires building evaluation datasets from production traces, support tickets, and manual testing, then running both correctness evaluations and open-ended criteria assessments. Governance concerns span all four phases: cost budgeting, tool access controls, audit trails, and human authorization checkpoints for high-stakes actions.

JIT warmup /dʒɪt ˈwɔːrmʌp/ Thời gian Node.js cần để tối ưu hóa code lúc runtime (Just-In-Time compilation). Compiler Go là AOT (ahead-of-time) nên không có warmup overhead.
Monorepo /ˈmɒnoʊriːpoʊ/ Kho code duy nhất chứa nhiều package/service — thường gặp ở big tech. TypeScript type checking trong monorepo rất tốn kém, nên 10× speedup có tác động lớn.
