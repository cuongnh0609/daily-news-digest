B1Copilot Studioエージェントを「バイブコーディング」で開発 — Claude Codeで設計図（ハーネス）を作りAgentic Loopを実装

📰 本文 + 和越対訳

2026年6月のCopilot Studio大型アップデートにより、エージェント開発のUIが自然言語ベースに刷新された。この記事では、エクスポートしたソリューションのXML・JSON構造を解析し、Claude Codeを用いてMarkdownベースの「ハーネス」（設計図）を生成してからエージェントを実装するアプローチを実践した記録を紹介している。かつてのパイプライン型設計から「推論→行動→観察」を繰り返すAgentic Loop型へと思想が転換したことで、バイブコーディング（自然言語指示でAIが実装するスタイル）との相性が格段に向上した。正解となる動作するソリューションを起点に最適化サイクルを回すことで、差分確認とバイト一致検証による高い再現性を実現している。このアプローチはCopilot Studio以外のエージェント開発にも応用可能な汎用的な手法として注目されている。

Sau bản cập nhật lớn của Copilot Studio vào tháng 6/2026, giao diện phát triển agent đã được làm mới theo hướng dùng ngôn ngữ tự nhiên. Bài viết mô tả quy trình phân tích cấu trúc XML/JSON của solution đã export, dùng Claude Code tạo ra "harness" (bản thiết kế dạng Markdown), rồi triển khai agent từ đó. Sự chuyển đổi từ pipeline truyền thống sang Agentic Loop (vòng lặp suy luận → hành động → quan sát) đã làm tăng đáng kể tính tương thích với vibe coding. Nhờ lấy solution hoạt động đúng làm gốc và chạy vòng lặp tối ưu hóa với xác minh byte-match, phương pháp đạt được tính tái hiện cao. Đây là kỹ thuật có thể áp dụng cho nhiều nền tảng agent khác ngoài Copilot Studio.

B3Deno 2.9正式リリース — 起動速度2倍・消費メモリ半減、WebViewでデスクトップアプリが作れる「Deno Desktop」登場

📰 本文 + 和越対訳

JavaScriptランタイム「Deno」のバージョン2.9が正式リリースされ、起動速度が従来の2倍に、消費メモリが半分に改善されたことが発表された。最大の目玉は新機能「Deno Desktop」で、既存のJavaScript・TypeScriptのWebアプリをそのままWindowsやmacOSのスタンドアロンアプリケーションとしてビルドできるようになった。内部にはWebViewを活用しており、Windows x64・macOS x64/arm64・Linux x64/arm64に対応している。CSSのインポート、npmパッケージのロックファイル対応など、Node.jsとの互換性向上も図られた。ElectronやTauriといった既存のデスクトップフレームワークと比較してセットアップが大幅に容易であり、TypeScript開発者がフロントエンドコードをそのままデスクトップに持ち込める点が評価されている。

Runtime JavaScript "Deno" phiên bản 2.9 ra mắt chính thức với tốc độ khởi động tăng gấp đôi và mức tiêu thụ bộ nhớ giảm một nửa. Tính năng nổi bật nhất là "Deno Desktop" — cho phép build ứng dụng standalone trên Windows/macOS từ code JavaScript/TypeScript web có sẵn, sử dụng WebView bên trong. Ngoài ra, Deno 2.9 cũng cải thiện tính tương thích với Node.js, hỗ trợ import CSS và lock file cho npm package. So với Electron hay Tauri, thiết lập Deno Desktop đơn giản hơn đáng kể, và được đánh giá cao vì developer TypeScript có thể đem thẳng frontend code lên desktop.
