B-03Python 3.15 ベータ1リリース——frozendict・遅延import・UTF-8デフォルト化など多数の新機能

📰 本文 + 和越対訳

Python 3.15のベータ1がリリースされ、複数の言語仕様強化と標準ライブラリの拡充が加わった。注目すべき変更点として、PEP 814によるイミュータブルな辞書型frozendictの追加、PEP 810による明示的な遅延importでアプリ起動時間の短縮が可能になった点が挙げられる。また、PEP 686によってテキストI/OのデフォルトエンコーディングがUTF-8に統一された。これはUTF-8ファイルを扱う際のコードをシンプルにする一方、Shift-JISなどのレガシーエンコーディングには明示的な指定が必要になるため、既存の日本語処理コードへの影響を確認しておく必要がある。さらにPEP 831でフレームポインタがデフォルト有効化され、プロファイリングや観測性の向上も図られている。

Python 3.15 Beta 1 đã được phát hành với nhiều cải tiến quan trọng. Điểm nổi bật gồm: PEP 814 bổ sung kiểu dict bất biến frozendict (dùng được làm key của set/dict), PEP 810 cho phép lazy import rõ ràng giúp giảm thời gian khởi động ứng dụng, và PEP 686 đặt UTF-8 làm encoding mặc định cho text I/O. Thay đổi sau sẽ giúp code xử lý file UTF-8 đơn giản hơn, nhưng đòi hỏi phải khai báo tường minh khi dùng encoding legacy như Shift-JIS — đây là điểm cần kiểm tra lại với code xử lý tiếng Nhật cũ. Ngoài ra PEP 831 kích hoạt frame pointers theo mặc định, cải thiện khả năng profiling và observability. Đây là bản beta, chưa dùng cho production.

C-02First Public macOS Kernel Memory Corruption Exploit on Apple M5 — MIE Bypass Achieved in 5 Days with AI Assistance

📰 Article (English)

Security researchers at Calif published the first publicly disclosed kernel memory corruption exploit targeting Apple Silicon M5 Macs running macOS 26.4.1. The exploit bypasses Memory Integrity Enforcement (MIE) — Apple's hardware-assisted memory safety mechanism built on ARM Memory Tagging Extensions (MTE), which Apple spent approximately five years and billions of dollars developing precisely to stop this category of attack. The full exploit chain achieves local privilege escalation from an unprivileged user to root via standard syscalls, with no special hardware modifications required. The attack was discovered on April 25, a working exploit was completed by May 1, and Apple was notified via in-person disclosure at Apple Park on May 3. Critically, AI assistance from a restricted model preview was cited as materially accelerating the research, compressing what might have been months of work into roughly five days. Apple's own research had claimed MIE "disrupts every known public exploit chain against modern iOS." The successful bypass challenges that assertion and signals that AI-human collaboration is now shortening the window between vulnerability discovery and weaponization.
