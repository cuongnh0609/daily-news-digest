📰 本文 + 和越対訳

複数のAIコーディングエージェントを並列で管理するために設計されたターミナルマルチプレクサ「herdr」がZennで注目を集めている。Rustで実装されたこのツールは、tmuxと同一のデフォルトキーバインディング（Ctrl+Bプレフィックス）を採用しており、移行コストが低い。最大の特徴は、エージェントのステータスを自動検出する仕組みで、「Blocked」「Working」「Done」「Idle」の4状態をサイドバーに常時表示する。さらにCLIとUNIXソケットAPIを通じて、エージェント自身がherdrを操作できる点がtmuxとの本質的な違いだ。GitHubインデックス経由のプラグインマーケットプレイスも提供されており、SSH接続やセッション永続化も標準機能として含まれている。AI駆動開発の現場では、複数エージェントの状態把握が生産性の鍵になりつつあり、こうしたツールへの関心は高まる一方だ。

Terminal multiplexer "herdr" được thiết kế đặc biệt để quản lý nhiều AI coding agent song song đang thu hút sự chú ý trên Zenn. Công cụ này được viết bằng Rust và giữ nguyên key binding mặc định của tmux (prefix Ctrl+B), nên chi phí chuyển đổi rất thấp. Điểm nổi bật nhất là cơ chế tự động phát hiện trạng thái của agent — hiển thị 4 trạng thái "Blocked", "Working", "Done", "Idle" liên tục trên thanh sidebar. Sự khác biệt cốt lõi so với tmux là agent có thể tự điều khiển herdr thông qua CLI và UNIX socket API, thay vì phải dùng `send-keys` thủ công. Plugin marketplace được index qua GitHub cũng được cung cấp, kèm theo SSH và session persistence như tính năng tiêu chuẩn. Anthropic released detailed documentation on Fable 5's cybersecurity safety architecture and introduced a new standardized scoring system called the Cyber Jailbreak Severity (CJS) scale. The framework classifies cybersecurity requests into four tiers: prohibited use (ransomware, malware authoring), high-risk dual use (penetration testing, exploit development), low-risk dual use (vulnerability scanning), and benign use (patch management, incident response). The CJS scale rates jailbreaks across four axes — capability gain, breadth of capability gain, ease of weaponization, and discoverability — producing ratings from CJS-0 (Informational) to CJS-4 (Critical). Unlike previous ad-hoc assessments, this framework provides a reproducible method for measuring how much a jailbreak actually advances attacker capabilities beyond freely available tools. Anthropic acknowledged that an expanded "safety margin" blocks some legitimate benign requests to prevent high-risk dual-use prompts from slipping through. A dedicated HackerOne bug-bounty program now accepts reports of discovered cyber jailbreaks, with rewards scaled to CJS severity. Security researchers working on red-teaming or AI model evaluation will find the published methodology directly applicable to benchmarking other frontier models.
