B1Claude CodeのPlanモードをループエンジニアリングで自動化——要件詰め・サブエージェント委譲・HTML確認を hook に組み込む

📰 本文 + 和越対訳

Claude Code の Plan モード（実装前に計画を立てるフェーズ）を、人間の介入コストを最小限に抑えながら活用するためのループエンジニアリング手法が Zenn に投稿された。著者は hook を使って、Plan モードに「質問による要件の深掘り」「実装のサブエージェントへの委譲」「計画の HTML 確認」という3つの自動処理を組み込んだ。特に AskUserQuestion hook を活用することで、Plan モードの初回プロンプト以降のタイピング負荷を大幅に削減できる点が注目されている。プランが reject された場合は自動再生成サイクルが走り、モデル・セッションの割り当てはハーネス寄りの設計で管理される。この手法により、単なるコーディングツールとしての Claude Code から、より自律的なエンジニアリングワークフローへの移行が実現しやすくなるという。

Một kỹ thuật "loop engineering" để tự động hóa chế độ Plan của Claude Code (giai đoạn lập kế hoạch trước khi triển khai) đã được đăng lên Zenn. Tác giả sử dụng hook để tích hợp 3 chức năng tự động vào Plan mode: "đào sâu yêu cầu bằng câu hỏi", "ủy quyền triển khai cho subagent", và "xác nhận kế hoạch qua HTML". Đặc biệt, việc sử dụng AskUserQuestion hook giúp giảm đáng kể gánh nặng gõ phím sau prompt đầu tiên của Plan mode. Khi kế hoạch bị từ chối, vòng lặp tái tạo tự động sẽ chạy, và việc phân bổ model/session được quản lý theo thiết kế gần với harness. Phương pháp này giúp chuyển đổi Claude Code từ công cụ coding đơn thuần sang workflow engineering tự chủ hơn.

B2Cloudflare Agents Week 2026——7つの新機能でAIエージェントを本番運用する完全ガイド
