📰 本文 + 和越対訳

Microsoftは「Microsoft Build 2026」において、次世代の量子チップ「Majorana 2」を発表した。同チップは量子ビットの信頼性を従来比で1,000倍向上させることに成功し、量子状態を最長1分間維持できる。これにより、実用的な量子コンピュータの実現目標を当初の2030年代から2029年に前倒しした。Majoranaはノーベル賞級の物理学者エットーレ・マヨラーナにちなんで命名されており、トポロジカル量子ビットと呼ばれる安定性の高い方式を採用している。量子コンピュータが実用化されれば、創薬・暗号・材料科学・AIトレーニングなど幅広い分野で革命的な変化をもたらすと期待されている。

C31-Click GitHub Token Theft via VSCode Bug — Malicious Jupyter Notebook Silently Installs Extension, Exfiltrates All Private Repo Access

📰 Article (English)

Security researcher Ammar Askar disclosed a critical vulnerability in VSCode's webview keyboard event handling that enables full GitHub token theft with a single click. The attack works by having a victim visit a github.dev URL pointing to a malicious repository. Inside that repo, a Jupyter notebook dispatches synthetic keyboard events that VSCode treats as real user input (Ctrl+Shift+A), triggering installation of a malicious workspace extension via a .vscode/extensions directory. The extension, loaded with skipPublisherTrust: true, silently exfiltrates the victim's GitHub OAuth token — which grants full access to all private repositories, not just the one being viewed. The attack chain is fully automated and works because github.dev lacks CSRF protection, meaning any external site can redirect victims there. For desktop VSCode, the attack requires cloning the repo first. Mitigation: clear github.dev site data in your browser. Askar published full disclosure simultaneously with notification because MSRC had previously silently patched his earlier report with no credit. The PoC is available on GitHub.
