📰 本文 + 和越対訳

米CognitionはAIコードエディタ「Windsurf」を「Devin Desktop」としてリニューアルした。Cognitionが独自開発したAIエージェント「Devin」と、2025年7月に買収したWindsurfを統合した形で、開発者が複数のAIエージェントを1つのインターフェースで管理できる統合開発環境（IDE）として提供される。新環境の最大の特徴は、オープンソースの「Agent Client Protocol（ACP）」に対応している点だ。これにより、AnthropicのClaude AgentやOpenAIのCodex、さらに企業が内製したACP準拠エージェントまでを、Devinと同じ画面からカンバン形式でシームレスに操作できる。新機能「Spaces」では、異なるエージェント間でセッション、ファイル、コンテキストが共有可能になった。既存のWindsurfユーザーにとっては通常のアップデートとして届き、プラン・料金・拡張機能はそのまま引き継がれる。キーバインドについてもVS Codeおよび旧Windsurfとの後方互換性が確保されており、移行コストは最小限に抑えられている。

C1OpenAI Launches Lockdown Mode: Cutting Off the "Lethal Trifecta" of Prompt Injection Attacks

📰 Article (English)

OpenAI has launched Lockdown Mode for ChatGPT, a security feature now rolling out to personal, paid, and business accounts. According to the official documentation, Lockdown Mode is designed to "help prevent the final stage of data exfiltration from a prompt injection attack by limiting outbound network requests." Security analysts describe prompt injection attacks through a framework called the "Lethal Trifecta": an AI system simultaneously has access to sensitive user data, is exposed to malicious third-party content, and retains the capability to exfiltrate that data to external destinations. By cutting off the third leg — the outbound request pathway — Lockdown Mode eliminates the condition that allows data theft to complete. Simon Willison, who covered this on his blog, highlights that this is an elegant defense because it relies on deterministic, rule-based blocking rather than LLM-evaluated safeguards that attackers might trick. The feature was initially previewed in February 2026 but has now become broadly available. Willison also notes that the very existence of Lockdown Mode implicitly acknowledges that ChatGPT's default configuration does not fully protect against sophisticated data exfiltration attacks — it's a significant admission baked into a product feature.

Speculative code /ˈspekjʊlətɪv/ Code viết "phòng xa" cho chức năng chưa có test yêu cầu — TDD thuần cấm điều này để tránh over-engineering.
Agent Skill /ˈeɪdʒənt skɪl/ Tập hướng dẫn (thường là markdown) dạy AI agent cách tiếp cận một loại tác vụ cụ thể — tương tự AGENTS.md hoặc Claude Code Skills.
Spec-driven development /spek drɪvən/ Phong cách phát triển bắt đầu từ specification (đặc tả), để spec dẫn dắt cả việc viết test lẫn code — phân biệt với code-then-test thông thường.
