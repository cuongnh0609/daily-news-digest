C-1ScarfBench: AI Agents Achieve Under 10% Success on Real-World Java Framework Migration

📰 Article (English)

IBM Research has released ScarfBench, an open benchmark that evaluates AI coding agents on cross-framework Java migration tasks spanning Spring, Jakarta EE, and Quarkus. The benchmark comprises 34 enterprise applications with 204 migration tasks totaling approximately 151,000 lines of code — far closer to real-world scale than academic toy problems. The headline finding is stark: even the strongest current AI agents achieve less than 10% behavioral success, meaning they can produce compilable code but fail to preserve the application's actual runtime behavior. Agents consistently struggle with dependency resolution, repeatedly revisiting configuration layers rather than following a linear migration path. IBM researchers also documented agent overconfidence: agents frequently self-report success when the migrated application is actually broken. The team identifies that "the biggest challenge in framework modernization is not translating Java code" but managing cascading dependency changes across configuration, infrastructure, and runtime environments — precisely the type of multi-hop reasoning at which current LLMs underperform. ScarfBench is fully open-source and provides leaderboards, datasets, and evaluation infrastructure, making it the first realistic agentic eval for enterprise Java modernization.

C-2Vercel's Andrew Qu: Agents Are a New Kind of Software Requiring New Primitives

📰 Article (English)

Vercel's Chief of Software Andrew Qu, speaking at Latent Space, argues that AI agents represent a fundamentally different category of software from traditional web applications — and that the industry has not yet fully internalized what this means for tooling and architecture. Unlike predictable web apps, agents require "different primitives for context, tools, resumability and long-running work." Vercel arrived at this understanding the hard way while building v0: they kept solving the same problems repeatedly until the patterns became obvious enough to standardize into a dedicated agent framework called eve. Qu identifies filesystem agents, skills (structured domain knowledge modules), and sandboxes as core building blocks of any agent system worth deploying in production. He draws a sharp distinction between agents that are good at "repetitive tasks that still require some reasoning" (contract review, data analysis) versus fully autonomous decision-making — and advocates for choosing the right level of human involvement for each task. A notable infrastructure implication: Vercel now serves agents machine-readable Markdown instead of HTML, recognizing that the future web must be simultaneously accessible to both humans and AI systems. The company is progressively making its own website, Slack workspace, and product dashboard agent-accessible.

Current AI, a nonprofit launched in February 2025 with $400 million in committed funding, has published the Open Source AI Gap Map v0.1 — the most comprehensive structured inventory of the open-source AI ecosystem to date.
