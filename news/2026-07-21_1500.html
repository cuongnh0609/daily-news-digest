📰 本文 + 和越対訳

2026年上半期（1〜6月）の家庭用エアコンの国内出荷額が前年同期比で21%超増加したことが明らかになった。背景には「エアコン2027年問題」がある。2027年に家庭用エアコンの冷媒として広く使われてきたR32（HFC系フロン）が一部規制対象となるため、消費者が旧式機種の早期買い替えを急いでいる。猛暑日の頻発も需要を後押ししており、量販店ではハイエンド機種を中心に品薄状態が続く。メーカー各社は生産ラインの増強を急いでいるが、半導体不足の影響で部品調達に遅れが生じている例も出ている。省エネ性能や新冷媒への対応を訴求した製品がシェアを伸ばしている。

B1仕様駆動開発の「仕様とコードのズレ」をハッシュで決定的に検出するCLIツール「artgraph」

📰 本文 + 和越対訳

仕様駆動開発（Spec-Driven Development）を実践するうえで長年の課題だった「仕様書とコードの乖離（ドリフト）」を、LLMの推論に頼らず決定的なハッシュベースの仕組みで検出するCLIツール「artgraph」が公開された。Spec KitやKiroが仕様書に振るフィンガープリントと同様の考え方を用い、仕様・ドキュメント・コード・テストの各ファイルのハッシュを管理することで、「コードが変わったのに仕様書が古いまま」「仕様を変えたのに実装が置き去り」といった状態を確実に検知する。CIに組み込めば、PRマージ前にドリフトを防ぐゲートとして機能する。作者はSpec KitやKiroに触発されており、仕様駆動開発ワークフローをよりロバストにするためのオープンソースの実装として位置づけている。

Một CLI tool mới tên "artgraph" vừa được công bố, giải quyết vấn đề lâu nay trong Spec-Driven Development (phát triển dựa trên đặc tả): sự "trôi dạt" (drift) giữa tài liệu đặc tả và code. Thay vì dựa vào LLM để phát hiện, artgraph sử dụng cơ chế dựa trên hash (fingerprint) — tương tự cách Spec Kit và Kiro gán fingerprint cho tài liệu đặc tả. Tool quản lý hash của từng file đặc tả, tài liệu, code và test, phát hiện chính xác các trạng thái như "code đã thay đổi nhưng spec vẫn cũ" hay "spec đã cập nhật nhưng implementation chưa theo kịp".
