01Claude Codeの新機能「/ultrareview」— エージェント集団によるマージ前の深層レビュー

📰 本文 + 和越対訳

Anthropic が Claude Code v2.1.86 で追加した research preview 機能「/ultrareview」は、マージ前のブランチや Pull Request をクラウド上のサンドボックス環境で複数のレビューエージェントに並列で分析させる仕組みだ。コマンドを引数なしで実行するとローカルブランチが束ねられ、GitHub リモートがなくても動作する。一方、/ultrareview <PR番号> と指定すれば該当 PR をマルチエージェントで精査できる。ユーザーが自分でエージェント群をオーケストレーションする必要はなく、バグ検出・設計レビュー・セキュリティ観点までを Claude 側が分担して並列実行する。課金はクラウドサンドボックスの利用分が加算されるため、使いどころを選ぶ必要がある。

Tính năng research preview `/ultrareview` mà Anthropic thêm vào Claude Code v2.1.86 cho phép chạy **nhiều review agent song song** trên sandbox cloud để phân tích branch hoặc Pull Request trước khi merge. Gọi không tham số sẽ bundle branch local lại — hoạt động được ngay cả khi không có GitHub remote. Nếu chỉ định `/ultrareview ` thì sẽ quét kỹ PR đó bằng multi-agent. User không phải tự orchestrate các agent — Claude tự phân công theo hướng bug detection, design review, security review và chạy song song. Do tính thêm phí cloud sandbox nên cần chọn đúng thời điểm dùng (merge lớn, PR phức tạp).
