📰 本文 + 和越対訳

AnthropicがAPIとして提供するAIエージェント実行基盤「Claude Managed Agents」が継続的なアップデートを重ね、2026年6月時点では開発者が独自のエージェントループやツール実行ロジックを実装することなく、Claudeにファイル読み取り・Web検索・コマンド実行などの複雑なタスクを委任できる状態になっている。最新の大きなアップデートとして、エージェントの動作パターンを記憶・圧縮する「Dreaming」機能、長期タスクに対応したスケジュール実行機能、さらにエージェントが独立したコンテキストウィンドウ内で別のエージェントに仕事を委任する「マルチエージェント・オーケストレーション」が加わった。「Vault（保管庫）」という機能では、スケジュール実行やCLIツールの認証情報をセキュアに管理できる。「Auto-Kans」機能により、開発者は「何をしてほしいか」を記述するだけでよく、エージェント自身が目標に向けて自律的に行動を積み重ね、途中で出力を自己評価しながら修正できるようになっている。これらの機能はAIエージェントを本番環境に組み込む際の基盤として注目を集めている。

Nền tảng thực thi AI agent "Claude Managed Agents" của Anthropic đang liên tục cập nhật, đến tháng 6/2026 cho phép developer giao phó các tác vụ phức tạp như đọc file, tìm kiếm web, chạy lệnh cho Claude mà không cần tự xây dựng agent loop hay tool execution logic. Các tính năng mới nổi bật bao gồm "Dreaming" — ghi nhớ và nén pattern hoạt động của agent; tính năng scheduled execution cho tác vụ dài hạn; và "Multi-agent orchestration" cho phép agent ủy quyền cho agent khác trong context window độc lập. "Vault" quản lý credentials cho scheduled deployment và CLI tools một cách bảo mật. Với "Auto-Kans", developer chỉ cần mô tả mục tiêu, agent tự chủ hành động và tự đánh giá output trong quá trình thực hiện.
