📰 本文 + 和越対訳

リハブフォージャパンのCTOである久良木氏が、Cloud Composer（Apache Airflow）で稼働していたデータ基盤を、Claude Codeを活用して一人で約2か月かけてCloud Run Jobsへ移行した事例を公開した。旧基盤は24時間稼働で月額600〜1,000ドルのコストがかかっていたが、実際の処理は1日数時間程度に過ぎなかった。移行後はCloud SchedulerとCloud Run Jobsの組み合わせで月額10〜20ドルまで圧縮し、94%のコスト削減を実現した。Claude Codeとの協働では、アーキテクチャの判断は人間が担い、実装タスクをClaude Codeに分担するという役割分担が功を奏した。41のタスク・50超のPR・約1万9,000行のコード（うちテスト行が約8,500行）という規模の成果が得られ、AIを活用したソロエンジニアリングの実践例として注目を集めている。

Trong quá trình cộng tác với Claude Code, con người chịu trách nhiệm quyết định kiến trúc, còn Claude Code đảm nhiệm các task triển khai. Kết quả gồm 41 task, hơn 50 PR và khoảng 19.000 dòng code (trong đó 8.500 dòng test), là ví dụ thực tiễn nổi bật về solo engineering với AI.
