B-01Claude Desktop Coworkをオンプレ企業でも——Amazon Bedrock経由でWindows展開する方法

📰 本文 + 和越対訳

AnthropicのClaude Desktopアプリが持つエージェント機能「Cowork」をAmazon Bedrock経由でWindows環境に展開する方法が、AWSの公式Zennアカウントによって詳細に解説された。Claude Desktopは通常、Anthropic APIに直接接続して動作するが、企業環境でのデータガバナンスやコスト管理を重視する場合、BedrockをバックエンドとしてIAMポリシーで細かくアクセス制御を行う構成が有効だ。 セットアップは管理者と利用者の2ロールで進める。管理者側ではBedrockのQuota確認とPlaygroundでの動作テスト、IAMポリシーの作成（bedrock:CallWithBearerTokenなどの権限付与）、IAMユーザーとベアラートークンの発行・配布を担う。利用者側では、Windowsの「仮想マシンプラットフォーム」機能の有効化（管理者権限が必要）、Claude Desktopのインストール、Bedrockの認証情報入力、さらにMCPサーバー（Brave Search・Chrome DevTools）の設定を行う。これにより、Claude CodeのエージェントコアをBedrock経由で動かしながら、Web検索や自動ブラウザ操作も可能になる企業向けセキュアな環境が実現する。

Tài khoản Zenn chính thức của AWS đã giới thiệu chi tiết cách deploy tính năng agent "Cowork" của Claude Desktop trên môi trường Windows thông qua Amazon Bedrock. Thay vì kết nối trực tiếp đến Anthropic API, cấu hình qua Bedrock cho phép kiểm soát truy cập chi tiết bằng IAM policy, phù hợp với yêu cầu quản trị dữ liệu của doanh nghiệp. Quá trình setup chia thành 2 vai trò: admin (kiểm tra Quota, tạo IAM policy với quyền `bedrock:CallWithBearerToken`, phát bearer token) và người dùng (bật tính năng Virtual Machine Platform trên Windows, cài Claude Desktop, nhập thông tin xác thực Bedrock, cấu hình MCP server). Kết quả là môi trường an toàn cho doanh nghiệp cho phép chạy agent Claude Code qua Bedrock, tích hợp tìm kiếm web và tự động hóa trình duyệt.
