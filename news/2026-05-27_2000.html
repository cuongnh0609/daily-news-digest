📰 本文 + 和越対訳

GitHub đã open-source extension Copilot cho IDE Eclipse, phát hành toàn bộ codebase theo giấy phép MIT với repository công khai trên GitHub. Quyết định này tiếp nối mô hình GitHub đã làm trước đó với các editor khác, tiếp tục xu hướng làm cho nội bộ của AI coding tool trở nên minh bạch và có thể mở rộng bởi cộng đồng. Bằng cách mở phần implementation cho Eclipse, developer có thể kiểm tra chi tiết kỹ thuật về cách các tính năng như code completion inline, chat và agentic workflow nhiều bước được xây dựng và tích hợp với kiến trúc plugin của Eclipse. GitHub mời đóng góp và phản hồi từ cộng đồng theo quy trình pull request và issue thông thường. Động thái này mang ý nghĩa đặc biệt với hệ sinh thái Java doanh nghiệp — Eclipse vẫn được dùng rộng rãi trong môi trường phát triển doanh nghiệp lớn tại các ngành ngân hàng, bảo hiểm và sản xuất, đặc biệt ở các thị trường như Nhật Bản nơi stack Java legacy phổ biến. Open-source extension cho phép team doanh nghiệp audit tích hợp Copilot cho yêu cầu bảo mật và compliance, chỉnh sửa hành vi theo chính sách nội bộ, và đóng góp cải tiến ngược lại upstream. Đây cũng là tín hiệu về chiến lược tổng thể của GitHub nhằm xây dựng lòng tin cộng đồng và đẩy nhanh áp dụng AI coding tool qua tính minh bạch.
Nếu bạn hoặc team đang dùng Eclipse cho Java enterprise (ví dụ Spring Boot, Jakarta EE), bây giờ có thể xem xét tích hợp Copilot mà không lo "hộp đen" — có thể audit code trước khi deploy trong môi trường có yêu cầu bảo mật nghiêm ngặt.
