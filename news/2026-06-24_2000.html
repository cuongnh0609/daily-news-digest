📰 本文 + 和越対訳

Google Cloudチームが2026年6月に提案した「OKF（Organized Knowledge Format）」は、AIエージェントに渡す社内ナレッジをマークダウン形式で統一的に管理するためのフォーマット仕様だ。ファイルの冒頭にYAML形式のメタデータ（type、title、description、tags、timestampなど）を配置するというシンプルなルールだが、これによりLLMによるRAG検索の精度が大幅に向上する。従来の「LLM Wiki」のような全自動ナレッジ管理だけでは属人化や検索精度の低下が生じやすく、統一されたメタデータ規則が必要だという課題意識から生まれた。OKFの設計哲学は「極力ルールを決めない」「フォーマットのみ定義する」「作成と利用を分離する」の3原則で、マークダウンの自由度を保ちながらエージェント親和性を実現している。

"OKF (Organized Knowledge Format)" do nhóm Google Cloud đề xuất tháng 6/2026 là đặc tả định dạng quản lý kiến thức nội bộ bằng Markdown nhằm tối ưu cho AI agent. Quy tắc cốt lõi rất đơn giản: đặt metadata YAML (type, title, description, tags, timestamp...) ở đầu file, nhưng điều này giúp cải thiện đáng kể độ chính xác khi LLM thực hiện tìm kiếm RAG. Ý tưởng xuất phát từ thực tế rằng chỉ dựa vào quản lý kiến thức tự động hoàn toàn như "LLM Wiki" dễ dẫn đến kiến thức bị cá nhân hóa quá mức và độ chính xác tìm kiếm giảm sút. Triết lý thiết kế của OKF gồm 3 nguyên tắc: "đặt ít quy tắc nhất có thể", "chỉ định nghĩa format", "tách biệt giữa tạo lập và sử dụng".

The research represents a meaningful step toward generalizable agentic AI that doesn't need task-specific simulation environments.

C-3Simon Willison: OPFS + Pyodide Test Harness — Running Python Tests Entirely in the Browser with No Server

📰 Article (English)

Simon Willison has published details on a novel browser-based testing approach that combines OPFS (Origin Private File System) with Pyodide to run a full Python test suite entirely within the browser environment, with no server required. OPFS provides persistent sandboxed storage directly accessible to web workers, while Pyodide runs a full CPython interpreter via WebAssembly. The combination allows Datasette's test suite to execute in a browser tab: Python files are loaded into OPFS, Pyodide imports them, and pytest runs against the in-memory SQLite database. The key engineering insight is using a dedicated web worker thread to run Pyodide — this prevents blocking the main UI thread during test execution, keeping the interface responsive. Willison built this to enable rapid iteration on Datasette Apps (a feature that lets users host custom HTML applications inside Datasette), allowing developers to test browser-specific code paths without spinning up a local server. The approach also has implications for AI agent testing workflows where browser-isolated environments offer a safer, cheaper alternative to full server deployments for testing web-facing code.
