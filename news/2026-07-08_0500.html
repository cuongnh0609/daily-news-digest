📰 Article (English)

Simon Willison released sqlite-utils 4.0 on July 7th, the first major version bump since 3.0 in November 2020. The release introduces three flagship features alongside a small set of breaking changes documented in an upgrade guide. The headline addition is database schema migrations: developers can now define migrations in Python files using the library's existing table.transform() method, which handles complex alterations that native SQLite cannot perform directly. Additional improvements include upserts using SQLite's native syntax, automatic primary key detection, immediate query execution with statement validation, and default column type inference for CSV imports.
