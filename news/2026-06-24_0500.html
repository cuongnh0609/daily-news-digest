B13つのAIエージェントを連携させた全自動開発パイプライン——Antigravity・Claude Code・Codexによる分業アーキテクチャ

📰 本文 + 和越対訳

Zennに掲載されたこの記事では、3つの異なるAIエージェントを役割分担させた全自動開発パイプラインのアーキテクチャが提案されている。中心に置かれるのは単一のAGENTS.mdファイルであり、これが全エージェントの行動規則を統一する。具体的には、Antigravity CLI（AGY）が「上流リサーチャー」としてターミナル操作やドキュメント収集を担い、Claude Codeが「実装ハブ」として実際のコーディングと統合を行い、Codex CLIが「下流オーディター」として別ベンダーの視点からセキュリティレビューを実施するという三層構造だ。コンテキスト管理にはGoogle NotebookLMとHeadroomを組み合わせ、膨大なドキュメントを外部知識ベースに逃がすことでトークン消費を抑制している。また、Recursive Self-Improvement（RSI）による自動ルール変更は禁止し、必ずピアレビューを経てからルールを追加・修正する「証拠に基づく変更」の仕組みも導入することで、エージェントが暴走しない品質保証の枠組みを実現している。

Bài viết trên Zenn đề xuất một kiến trúc pipeline phát triển hoàn toàn tự động bằng cách phối hợp ba AI agent khác nhau, điều phối bởi một file AGENTS.md trung tâm. Antigravity CLI đóng vai "researcher thượng nguồn" thu thập tài liệu, Claude Code là "implementation hub" thực hiện coding và tích hợp, còn Codex CLI là "auditor hạ nguồn" thực hiện security review từ góc nhìn vendor khác. Để quản lý context, hệ thống kết hợp Google NotebookLM và Headroom để đưa tài liệu ra knowledge base bên ngoài, tiết kiệm token. Đặc biệt, RSI (Recursive Self-Improvement) bị cấm — mọi thay đổi rule đều phải qua peer review — tạo cơ chế đảm bảo chất lượng để agent không bị "chạy loạn".
