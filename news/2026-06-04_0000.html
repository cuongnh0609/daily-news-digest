B-01Claude Codeのセッション履歴をfzfで高速検索する「ccsession」ツールを自作した

📰 本文 + 和越対訳

Claude Codeを日常的に使い込んでいると、過去のセッション履歴に戻りたい場面が頻繁に発生する。しかし標準の履歴ブラウジング機能はやや不便で、検索性に乏しい。このZenn記事の著者は、ターミナル上でfzf（コマンドラインファジー検索ツール）を組み合わせた独自ツール「ccsession」を開発し、Claude Codeのセッションログを瞬時にプレビュー・検索できる環境を実現した。ツールはシェルスクリプトで実装されており、fzfのプレビューウィンドウに会話の冒頭数行を表示し、選択した瞬間にそのセッションをClaude Codeで再開できる仕組みを備える。使い勝手は従来の履歴ブラウジングに比べて大幅に向上したとのことで、Hookと組み合わせることでセッション終了時の自動タグ付けも可能。Claude Code Hooksの活用例として注目されており、Zennでのいいね数が急上昇中。

Khi sử dụng Claude Code hàng ngày, nhu cầu quay lại các phiên làm việc cũ phát sinh rất thường xuyên. Tuy nhiên, tính năng duyệt lịch sử tiêu chuẩn còn khá bất tiện và thiếu khả năng tìm kiếm. Tác giả bài viết trên Zenn đã tự xây dựng công cụ "ccsession" tích hợp `fzf` (fuzzy finder) để tìm kiếm và preview session log của Claude Code ngay trong terminal. Công cụ viết bằng shell script, hiển thị vài dòng đầu mỗi cuộc hội thoại trong cửa sổ preview của fzf, và cho phép mở lại session đó trong Claude Code ngay lập tức sau khi chọn. Kết hợp với Claude Code Hooks, người dùng còn có thể tự động gắn tag cho session khi kết thúc. Bài viết đang nhận được lượng "like" tăng nhanh trên Zenn.
