📰 本文 + 和越対訳

SonicGardenのmaedana氏が、Claude Codeのシングルセッション運用から複数セッション並列化へ至るまでの試行錯誤を詳細に報告した。Claude Codeの開発者Boris Cherny氏の並列ワークフローを見て触発され、同一リポジトリへの並列実行にはgit worktreeを活用し、別リポジトリへは単純に複数セッションを立ち上げるというアプローチを採用。課題として浮上したのは、複数セッションの状態可視化の困難さ、承認割り込みによる集中の断絶、コンテキスト切り替えコストの増大だった。この問題を解決するため、筆者はリアルタイムでセッション状態を表示するデスクトップウィジェット「claudeye」と、tmuxベースのマルチセッション管理ダッシュボード「crmux」という2つのツールを自作。副次効果として、Claude Codeが個人開発のハードルを下げ、放置していた個人プロジェクトへの熱意が再燃したとも述べている。

Kỹ sư maedana từ SonicGarden chia sẻ hành trình chuyển đổi từ việc dùng một session Claude Code sang chạy nhiều session song song. Để giải quyết, tác giả tự xây dựng hai công cụ: "claudeye" (widget desktop hiển thị trạng thái session realtime) và "crmux" (dashboard quản lý đa session dựa trên tmux). Một hiệu ứng phụ tích cực: Claude Code giúp hạ ngưỡng để bắt đầu dự án cá nhân, khơi lại niềm đam mê với các side project.

B2AGENTS.mdを「正本」とする設計——Claude CodeとCodexが同一リポジトリで衝突しないために

📰 本文 + 和越対訳
