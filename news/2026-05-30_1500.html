B1Claude Code Dynamic Worflowsを触ってみた——数百の並列サブエージェントが大規模タスクをこなす新アーキテクチャ

📰 本文 + 和越対訳

Claude Code v2.1.154で登場した「Dynamic Workflows」は、Claudeがオーケストレーションスクリプトを動的に生成し、数十〜数百のサブエージェントを並列で動かすことで、大規模タスクに対応する新たな実行モデルだ。従来の「1エージェントがシーケンシャルに作業」という流れと異なり、Dynamic Workflowsでは並列処理・独立した検証フェーズ・進捗の永続的追跡・エージェント同士による相互レビューという4つの柱で構成されている。実際のテストでは40エージェントが約40分稼働し、3.3Mトークンを消費しながら71件の問題を発見し、そのうち41%を相互検証で除外したという。また、Anthropicが買収したBun社のケースでは、750,000行のZig→Rustへの移植を11日間で完了させたことが実績として紹介されている。有効化は「ultracode」モードを選択するか、直接Dynamic Workflowsをリクエストする形で行う。

"Dynamic Workflows" xuất hiện trong Claude Code v2.1.154 là mô hình thực thi mới, cho phép Claude tự động tạo script orchestration để điều phối hàng chục đến hàng trăm subagent chạy song song xử lý các tác vụ quy mô lớn. Khác với luồng làm việc tuần tự của một agent đơn lẻ, Dynamic Workflows được xây dựng trên bốn trụ cột: xử lý song song, giai đoạn xác minh độc lập, theo dõi tiến độ bền vững và các agent tự kiểm tra lẫn nhau. Trong một bài test thực tế, 40 agent hoạt động trong ~40 phút, tiêu thụ 3,3M token, phát hiện 71 vấn đề và loại bỏ 41% qua kiểm tra chéo. Bun (công ty được Anthropic mua lại) đã sử dụng Dynamic Workflows để chuyển đổi 750.000 dòng code từ Zig sang Rust chỉ trong 11 ngày. Tính năng được kích hoạt qua chế độ "ultracode" hoặc bằng cách yêu cầu trực tiếp.
